# Title 标题

### 默认标题 size = l1

::: demo

```html
<template>
  <dp-title title="标题内容" />
</template>
```

:::

### 小标题 size = l2

::: demo

```html
<template>
  <dp-title title="标题内容size:l2" size="l2" />
</template>
```

:::

| 参数  | 说明         | 类型   | 可选值 | 默认值 |
| ----- | ------------ | ------ | ------ | ------ |
| title | 标题内容文本 | String | --     | --     |
| size  | 标题大小     | String | l1/l2  | l1     |
